# UART Transmitter and Receiver in Verilog

## Overview
This project implements a UART (Universal Asynchronous Receiver Transmitter) communication module using Verilog HDL. UART is widely used for serial communication between digital systems.

## Features
- UART Transmitter (TX)
- UART Receiver (RX)
- Start bit and stop bit implementation
- Configurable baud rate
- Serial data transmission and reception

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave

## Project Structure
uart_tx.v   -> UART transmitter module
uart_rx.v   -> UART receiver module
uart_tb.v   -> Testbench for simulation
